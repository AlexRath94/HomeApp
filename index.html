NFC-Tags beschreiben
Android (Chrome): Aufgabe antippen → „Tag jetzt beschreiben“ → NFC-Tag an die Rückseite des Handys halten. Fertig.

iPhone (mit NFC Tools):
  1. Aufgabe antippen → „Link kopieren“
  2. NFC Tools öffnen → Schreiben → Datensatz hinzufügen → URL/Webadresse
  3. Link einfügen → Schreiben → Tag anhalten
Scannen im Alltag: iPhone (XS oder neuer) einfach mit eingeschaltetem Bildschirm oben an den Tag halten — der Link öffnet sich automatisch. Android: Tag an die Rückseite halten (NFC muss in den Einstellungen aktiv sein). Günstige NTAG213-Tags reichen völlig.
Daten & Geräte
Die Aufgaben werden lokal im Browser dieses Geräts gespeichert. Mit Exportieren / Importieren kannst du ein Backup anlegen oder die Liste auf ein zweites Handy übertragen.

iPhone-Hinweis: NFC-Tags öffnen immer Safari. Nutze die App dort am besten als Lesezeichen — ein Symbol auf dem Home-Bildschirm hätte einen getrennten Speicher und würde die Häkchen nicht sehen.
